# Deno Payment Wallet

## Overview
A secure payment wallet application built with Deno, React, and SQLite.

## Features
- User Registration
- Secure Authentication
- Send Payments
- Accept Payments
- Wallet Balance Management

## Prerequisites
- Deno
- Modern web browser

## Installation
1. Run with Deno
